# Sprint 6 — Projeto Integrado 1

Projeto desenvolvido durante o bootcamp de Data Analytics da TripleTen.

## Objetivo

Analisar dados históricos da indústria de videojogos para identificar padrões de sucesso e apoiar estratégias de marketing e planeamento para 2017.

## Descrição

Neste projeto foram realizados:
- limpeza e pré-processamento de dados;
- análise exploratória de dados;
- análise de vendas globais de videojogos;
- comparação de plataformas e géneros;
- análise de avaliações de críticos e usuários;
- teste de hipóteses estatísticas;
- criação de gráficos e visualizações de dados.

## Tecnologias utilizadas

- Python
- Pandas
- NumPy
- Matplotlib
- SciPy
- Jupyter Notebook
